# Industrial Logbook OCR

OCR pipeline for extracting handwritten industrial process logbooks from PDF scans.

## Features

- PDF to image conversion
- Table detection
- Grid alignment
- Handwritten OCR
- EasyOCR + Tesseract hybrid processing
- Excel export

## Main Script

python ocr_production_run.py

## Project Structure

Diagnostics/           -> experiments and debugging
Debugging Utilities/   -> helper scripts
ocr_production_run.py  -> production OCR pipeline
